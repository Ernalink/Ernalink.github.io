# Ernalink ブランドアセット

Ernalink 公式ワードマーク（2026-07-15 確定版）の正規置き場です。
サイト内からの参照はすべて `/assets/brand/...` の絶対パスを使います。

## ブランド確定情報

| 項目 | 値 |
|---|---|
| ワードマーク a | `ERNALINK` ＋ 赤い終止符ドット（**対外正式**用） |
| ワードマーク b2 | `ERNALINK` の R の空洞に赤ドット（**顔になる場所**用） |
| Navy | `#0B1323` |
| Red | `#C43D3D` |
| Light（暗背景の文字色） | `#EDF1F8` |
| 使い分け | ヘッダー＝b2 ／ フッター＝a ／ 極小（ファビコン等）＝赤丸単体 |
| 本文中の屋号表記 | テキストの「Ernalink」のまま（ロゴに置換しない） |

SVG は文字アウトライン化済み・背景透過。`_navy`＝明るい背景用（紺文字）、`_light`＝暗い背景用（明文字）。

## 仕様

- 書体　　: Montserrat Light（アウトライン化済・フォント不要）
- 字間　　: 0.22em（220/1000units）
- カラー　: Navy `#0B1323` ／ Light `#EDF1F8` ／ Red `#C43D3D`
- 赤ドット: 半径 78units の共通エレメント
  - a  = ベースライン上・K の後 gap 60units
  - b2 = R カウンターの幾何中心（bbox 173-603 × 274-654）
- 背景　　: 透過（背景色は使用側で敷く）

## 使い分け（確定ルール）

- a  : 対外正式（フッター・請求書・提案書・メール署名）
- b2 : 顔になる場所（サイトヘッダー・名刺表・プロフィール）
- 極小: b2 の点が潰れるサイズでは素の文字 or 赤丸単体で代用
